<objective>
Migrate per-operation entry/exit `Effect.logInfo` calls to `Effect.logDebug` across all services and atoms, establishing the three-level convention: Debug (method entry/exit), Info (lifecycle/construction), Error (failures).

Purpose: Per D-11 and D-14, per-operation logs flood the Info channel. Moving them to Debug means dev builds still see everything (DevLogger shows all levels per D-13), but production file logging only captures errors. This creates a clean signal-to-noise ratio.

Output: 8 files updated with correct log level usage.
</objective>
