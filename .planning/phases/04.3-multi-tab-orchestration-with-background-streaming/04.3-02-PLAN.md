    - Remove the `form` object entirely (no more `useForm`)
    - Replace `<form.Field>` with a plain `<Input>` that reads `value={draft}` and writes `onChange={(e) => setDraft(e.target.value)}`
    - On submit handler: check `draft.trim()`, call `sendMessage({ tabId: tabKey, prompt: draft.trim() })`, then `setDraft("")`
    - Remove `form.reset()` call

    **2. Add scroll container ref:**
    - Add a ref for the scroll container div: `const scrollContainerRef = useRef<HTMLDivElement>(null)`
    - Attach `ref={scrollContainerRef}` to the `<div className="flex-1 space-y-3 overflow-y-auto p-4">` element
    - Remove the `messagesEndRef` div at the bottom of the scroll area (no longer needed)
    - Remove the `messagesEndRef` useRef declaration

    **3. Read/write scroll atoms:**
    - Read auto-scroll state: `const autoScroll = useAtomValue(autoScrollAtom(tabKey))`
    - Set auto-scroll: `const setAutoScroll = useAtomSet(autoScrollAtom(tabKey))`
    - Set scroll position: `const setScrollPosition = useAtomSet(scrollPositionAtom(tabKey))`
    - Read scroll position for restore: `const savedScrollPos = useAtomValue(scrollPositionAtom(tabKey))`
    - Set unread: `const setUnread = useAtomSet(unreadAtom(tabKey))`

    **4. Smart scroll threshold constant:**
    ```typescript
    const AT_BOTTOM_THRESHOLD = 32;
    ```
    Define as a module-level constant (outside the component), matching UI-SPEC 32px threshold.

    **5. Scroll event handler:**
    - Replace the existing useEffect that scrolls to bottom. Instead, add a scroll event listener on the scroll container:
    ```typescript
    useEffect(() => {
      const el = scrollContainerRef.current;
      if (!el) return;
      let rafId = 0;
      const onScroll = () => {
        cancelAnimationFrame(rafId);
        rafId = requestAnimationFrame(() => {
          const atBottom = el.scrollHeight - el.scrollTop - el.clientHeight < AT_BOTTOM_THRESHOLD;
          setAutoScroll(atBottom);
          setScrollPosition(el.scrollTop);
          if (atBottom) {
            setUnread(false);
          }
        });
      };
      el.addEventListener("scroll", onScroll, { passive: true });
      return () => {
        cancelAnimationFrame(rafId);
        el.removeEventListener("scroll", onScroll);
      };
    }, [tabKey, setAutoScroll, setScrollPosition, setUnread]);
    ```
    Note: React Compiler handles the dependency optimization. List all used setters in deps for correctness.

    **6. Restore scroll position on tab switch:**
    ```typescript
    useEffect(() => {
      const el = scrollContainerRef.current;
      if (!el) return;
      // Restore saved position on tab switch (D-13, D-15 instant)
      el.scrollTop = savedScrollPos;
    }, [tabKey, savedScrollPos]);
    ```
    This runs when `tabKey` changes, restoring the saved scroll position. Since savedScrollPos is an atom value, it captures the position from when the user was last on this tab.

    **7. Auto-scroll on new content:**
    ```typescript
    useEffect(() => {
      if (!autoScroll) return;
      const el = scrollContainerRef.current;
      if (!el) return;
      // Auto-scroll to bottom when new content arrives and auto-scroll is enabled (D-14)
      el.scrollTop = el.scrollHeight;
    }, [messages, streamingText, autoScroll]);
    ```
    This replaces the old `messagesEndRef.scrollIntoView` approach. Only scrolls when `autoScroll` is true.

    **8. Remove old scroll logic:**
    - Remove the `scrollRafRef` useRef
    - Remove the old useEffect that used `messagesEndRef.current?.scrollIntoView`

    **9. Keep the form submit handler:**
    ```typescript
    const handleSubmit = (e: React.FormEvent) => {
      e.preventDefault();
      e.stopPropagation();
      const trimmed = draft.trim();
      if (!trimmed || isStreaming) return;
      sendMessage({ tabId: tabKey, prompt: trimmed });
      setDraft("");
    };
    ```
    Attach to `<form onSubmit={handleSubmit}>`. Remove the `<form.Field>` wrapper entirely. Use a plain `<Input>` element:
    ```tsx
    <Input
      autoFocus
      disabled={isStreaming}
      onChange={(e) => setDraft(e.target.value)}
      placeholder={isStreaming ? "Waiting for response..." : "Message..."}
      value={draft}
    />
    ```
