## CWD Threading

| Option | Description | Selected |
|--------|-------------|----------|
| Per-tab cwdAtom | Add cwdAtom = Atom.family to chat.ts. Populated on tab activation. sendMessageAtom reads ctx(cwdAtom(tabId)) directly — no component changes needed. | ✓ |
| Look up from projectsAtom | sendMessageAtom finds the tab's cwd by searching projectsAtom sessions. No new atom, but couples chat.ts to sidebar.ts and requires iterating project/session arrays on every send. | |
