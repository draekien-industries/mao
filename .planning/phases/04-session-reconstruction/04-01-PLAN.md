    **2. Implement `getBySessionWithMeta` in `src/services/database/event-store/service.ts`:**

    Add import of `StoredEventWithMeta` from `./schemas`. Add the method implementation between `getBySession` and `purgeSession`. It uses the same SQL query as `getBySession` but returns `StoredEventWithMeta` objects:

    ```typescript
    const getBySessionWithMeta = (sessionId: string) =>
      Effect.gen(function* () {
        const rows = yield* sql<EventRow>`
          SELECT id, session_id, sequence_number, event_type, event_data, created_at
          FROM events
          WHERE session_id = ${sessionId}
          ORDER BY sequence_number ASC
        `;

        return yield* Effect.forEach(rows, (row) =>
          decodeStoredEvent(row.event_data).pipe(
            Effect.map((event) => ({
              createdAt: row.created_at,
              event,
              sequenceNumber: row.sequence_number,
            })),
            Effect.mapError(
              (cause) =>
                new DatabaseQueryError({
                  cause: String(cause),
                  message: `Failed to decode event ${row.id} for session ${sessionId}`,
                }),
            ),
          ),
        );
      }).pipe(
        Effect.mapError((cause) =>
          cause instanceof DatabaseQueryError
            ? cause
            : new DatabaseQueryError({
                cause: String(cause),
                message: `Failed to query events with meta for session ${sessionId}`,
              }),
        ),
        Effect.annotateLogs(annotations.operation, "getBySessionWithMeta"),
      );
    ```

    Update the return object to include `getBySessionWithMeta`:
    ```typescript
    return { append, getBySession, getBySessionWithMeta, purgeSession };
    ```
