<objective>
Implement the SessionReconstructor Effect service using TDD. This service folds stored events from EventStore.getBySessionWithMeta into a ReconstructedSession containing typed ChatMessage objects.

Purpose: This is the core read-path logic for Phase 4 -- it transforms raw persisted events back into the conversation state that the renderer needs. TDD is used because the fold logic has well-defined input/output contracts (event types -> ChatMessage array) and multiple edge cases (empty sessions, incomplete sessions, skipped event types).

Output: A fully tested SessionReconstructor service ready to be wired into RPC handlers (Plan 03).
</objective>

  <behavior>
    - reconstruct("session-1") with SystemInitEvent + UserMessageEvent + AssistantMessageEvent + ResultEvent -> ReconstructedSession { sessionId: session_id from SystemInitEvent, messages: [user ChatMessage, assistant ChatMessage] }
    - reconstruct("session-1") with SystemInitEvent only -> ReconstructedSession { sessionId: extracted_id, messages: [] }
    - reconstruct("session-1") with SystemInitEvent + UserMessageEvent (no assistant response) -> ReconstructedSession { sessionId: extracted_id, messages: [user ChatMessage] }
    - reconstruct("session-1") with SystemInitEvent + UserMessageEvent + AssistantMessageEvent + UserMessageEvent + AssistantMessageEvent -> ReconstructedSession with 4 messages in order
    - reconstruct("session-1") with SystemRetryEvent present -> that event is skipped (not in messages)
    - reconstruct("session-1") with UnknownEvent present -> that event is skipped
    - Each ChatMessage has id = row.sequenceNumber and createdAt = row.createdAt from StoredEventWithMeta
    - reconstruct("nonexistent") where getBySessionWithMeta returns [] -> ReconstructedSession { sessionId: "nonexistent", messages: [] }
  </behavior>

    **Service implementation (`service.ts`):**

    ```typescript
    import { Effect, Layer } from "effect";
    import { extractAssistantText } from "@/lib/extract-assistant-text";
    import { isAssistantMessage, isSystemInit } from "../../claude-cli/events";
    import { annotations } from "../../diagnostics";
    import { isUserMessage } from "../event-store/schemas";
    import { EventStore } from "../event-store/service-definition";
    import { ChatMessage, ReconstructedSession } from "./schemas";
    import { SessionReconstructor } from "./service-definition";

    export const makeSessionReconstructorLive = () =>
      Layer.effect(
        SessionReconstructor,
        Effect.gen(function* () {
          const eventStore = yield* EventStore;

          const reconstruct = (sessionId: string) =>
            Effect.gen(function* () {
              const rows = yield* eventStore.getBySessionWithMeta(sessionId);

              let extractedSessionId = sessionId;
              const messages: Array<ChatMessage> = [];

              for (const row of rows) {
                if (isSystemInit(row.event)) {
                  extractedSessionId = row.event.session_id;
                } else if (isUserMessage(row.event)) {
                  messages.push(
                    new ChatMessage({
                      content: row.event.prompt,
                      createdAt: row.createdAt,
                      id: row.sequenceNumber,
                      role: "user",
                    }),
                  );
                } else if (isAssistantMessage(row.event)) {
                  messages.push(
                    new ChatMessage({
                      content: extractAssistantText(row.event),
                      createdAt: row.createdAt,
                      id: row.sequenceNumber,
                      role: "assistant",
                    }),
                  );
                }
                // ResultEvent, SystemRetryEvent, UnknownEvent -> skipped (D-09)
              }

              return new ReconstructedSession({
                messages,
                sessionId: extractedSessionId,
              });
            }).pipe(
              Effect.annotateLogs(annotations.operation, "reconstruct"),
              Effect.annotateLogs(annotations.sessionId, sessionId),
            );

          return { reconstruct };
        }).pipe(
          Effect.annotateLogs(annotations.service, "session-reconstructor"),
        ),
      );
    ```

    **Test pattern:**

    Mock `EventStore` via `Layer.succeed(EventStore, mockImpl)` following the pattern from existing tests. Construct `StoredEventWithMeta` arrays with test data covering each behavior case. Run `SessionReconstructor.reconstruct` via `Effect.provide(testLayer)`.

    The mock EventStore needs all 4 methods: `append`, `getBySession`, `getBySessionWithMeta`, `purgeSession`. Only `getBySessionWithMeta` needs real logic (return canned data based on sessionId). The others return `Effect.succeed(...)` with dummy values.
