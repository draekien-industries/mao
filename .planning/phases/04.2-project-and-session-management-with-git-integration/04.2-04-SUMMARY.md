## Decisions Made

- Used `Effect.Effect.Success<typeof _makeClient>` to infer the merged RPC client type, avoiding complex generic type extraction from RpcGroup
- Changed `activeTabIdAtom` from `string` ("tab-1") to `number | null` since real tab IDs are SQLite integers
- Used `Effect.sync` for `setActiveTabAtom` since it only sets atom values synchronously (no yields)
- Fixed spinner.tsx props to use simple `className` instead of full SVG element props for HugeiconsIcon compatibility
