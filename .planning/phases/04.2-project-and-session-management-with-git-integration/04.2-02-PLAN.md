    **src/services/git/service.ts:**
    Export `makeGitServiceLive` factory returning `Layer.effect(GitService, Effect.gen(...))`:

    1. Yield `CommandExecutor.CommandExecutor` from the context.

    2. Create a private `runGitCommand(args: ReadonlyArray<string>, cwd: string)` helper:
       - `let command = Command.make("git", ...args)`
       - `command = Command.workingDirectory(command, cwd)`
       - `const process = yield* CommandExecutor.start(command)` (use the yielded executor)
       - Collect stdout via `Stream.decodeText() |> Stream.runFold("", (a, s) => a + s)`
       - Check `process.exitCode`; if non-zero, collect stderr similarly, yield GitOperationError
       - Return `stdout.trim()`

    3. Implement methods:

       **listBranches(cwd):**
       `runGitCommand(["branch", "--list", "--format=%(refname:short)"], cwd)`
       Split output by "\n", filter empty lines.

       **getCurrentBranch(cwd):**
       `runGitCommand(["rev-parse", "--abbrev-ref", "HEAD"], cwd)`

       **getRepoName(cwd):**
       ```typescript
       import path from "node:path";
       runGitCommand(["rev-parse", "--show-toplevel"], cwd)
         .pipe(Effect.map((topLevel) => path.basename(topLevel)))
       ```

       **isGitRepo(cwd):**
       `runGitCommand(["rev-parse", "--is-inside-work-tree"], cwd)`
       Map to `output === "true"`. Catch GitOperationError and return false.

       **listWorktrees(cwd):**
       `runGitCommand(["worktree", "list", "--porcelain"], cwd)`
       Parse the porcelain output: blocks separated by blank lines, each block has lines like:
       - `worktree /path/to/worktree`
       - `HEAD abc123`
       - `branch refs/heads/branch-name`
       - (or `detached` instead of branch)

       Parse into Worktree objects. Branch field: strip `refs/heads/` prefix. If `detached`, branch is null.

       **createWorktree(cwd, branchName, basePath):**
       ```typescript
       const worktreePath = path.join(basePath, branchName);
       // Check if branch exists first
       const branches = yield* listBranches(cwd);
       const branchExists = branches.includes(branchName);
       const args = branchExists
         ? ["worktree", "add", worktreePath, branchName]
         : ["worktree", "add", worktreePath, "-b", branchName];
       yield* runGitCommand(args, cwd);
       return worktreePath;
       ```

       **removeWorktree(cwd, worktreePath):**
       `runGitCommand(["worktree", "remove", worktreePath], cwd).pipe(Effect.asVoid)`

    4. Annotate with `Effect.annotateLogs(annotations.service, "git")`.

    - getRepoName: git output "/home/user/repos/mao\n" -> "mao"

    - removeWorktree: calls git worktree remove
    - Error case: non-zero exit code produces GitOperationError

    **src/services/dialog/service.ts:**
    Export `makeDialogServiceLive` factory returning `Layer.effect(DialogService, Effect.gen(...))`:

    ```typescript
    import { dialog } from "electron";
    import { Effect, Layer, Option } from "effect";
    import { annotations } from "../diagnostics";
    import { DialogError } from "./errors";
    import { DialogService } from "./service-definition";

    export const makeDialogServiceLive = () =>
      Layer.succeed(
        DialogService,
        {
          openDirectory: () =>
            Effect.tryPromise({
              try: () =>
                dialog.showOpenDialog({
                  properties: ["openDirectory"],
                  title: "Select Project Directory",
                }),
              catch: (cause) =>
                new DialogError({
                  message: String(cause),
                  operation: "openDirectory",
                }),
            }).pipe(
              Effect.map((result) =>
                result.canceled
                  ? Option.none<string>()
                  : Option.some(result.filePaths[0]),
              ),
              Effect.annotateLogs(annotations.operation, "openDirectory"),
              Effect.annotateLogs(annotations.service, "dialog"),
            ),
        },
      );
    ```

    Note: DialogService uses `Layer.succeed` (not `Layer.effect`) since it doesn't need to yield any dependencies -- `dialog` is imported directly from Electron (main process only). No unit test for DialogService since it wraps Electron's native dialog which cannot be imported in vitest (Electron ABI mismatch, same reason TabStore tests use mocks).
